---

## 🧠 Visão Geral

Este projeto consiste na criação de um formulário web moderno e funcional, com foco em boas práticas de estruturação, semântica e estilização.

A proposta é simular um formulário real, aplicando conceitos essenciais de front-end sem o uso de frameworks ou bibliotecas externas.

---

## 🎯 Objetivos do Projeto

* Praticar HTML semântico
* Aplicar estilização com CSS puro
* Criar um layout limpo e organizado
* Desenvolver noções de responsividade
* Construir base sólida para projetos futuros

---

## 🚀 Funcionalidades

* Campos de entrada estruturados 
* Organização visual clara e intuitiva
* Feedback básico com validação HTML
* Layout centralizado e agradável


## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização e layout
